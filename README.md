# Day 1 – Java Basics & Control Statements

This repository contains Java programs implemented as part of Day 1 lab work.  
The focus was on understanding Java fundamentals, operators, conditional statements, and loops.

## Section 1 – Basic Programs

### Topics Covered
- Arithmetic operators
- If-else statements
- For loop
- While loop
- Basic mathematical calculations
- Number manipulation

### Programs Implemented
1. Area of a Circle
2. Swap Two Numbers (without third variable)
3. Simple Calculator
4. Even or Odd
5. Largest of Three Numbers
6. Grade Calculator
7. Print Numbers from 1 to N
8. Sum of First N Natural Numbers
9. Multiplication Table
10. Reverse a Number

## Section 2 – Conditional & Loop-Based Programs

### Topics Covered
- Mathematical formulas
- Conditional checking
- Leap year logic
- Character checking
- Factorial using loop
- Digit manipulation

### Programs Implemented
1. Celsius to Fahrenheit Conversion
2. Simple Interest Calculation
3. Positive, Negative or Zero Check
4. Find Remainder
5. Leap Year Checker
6. Vowel or Consonant
7. Factorial of a Number
8. Count Digits
9. Sum of Digits
10. Print Even Numbers (1–100)
